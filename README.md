# 🎉 День Батька — Веб-сайт для Тата

Прекрасний, адаптивний веб-сайт на Python Flask, присвячений Дню батька. Сайт має преміальний дизайн з темною темою, плавними анімаціями та інтерактивними елементами.

## ✨ Особливості

- **Головна сторінка** з ефектом паралаксу та плавною fade-in анімацією
- **Галерея спогадів** з картками, що перевертаються (flip-cards)
- **Інтерактивна секція "Послання"** з ефектом друку тексту (typing effect)
- **Святкові конфеті** при кліку на кнопки та картки
- **Темна елегантна тема** з сучасними градієнтами та тінями
- **Адаптивний дизайн** для всіх пристроїв
- **Scroll animations** за допомогою AOS (Animate On Scroll)
- **Tailwind CSS** для сучасної стилізації

## 📁 Структура проєкту

```
site-for-dad/
├── app.py                 # Головний файл Flask додатку
├── requirements.txt       # Залежності Python
├── README.md             # Цей файл
├── templates/            # HTML шаблони
│   ├── base.html        # Базовий шаблон
│   └── index.html       # Головна сторінка
└── static/              # Статичні файли
    ├── css/
    │   └── style.css    # Кастомні стилі CSS
    └── js/
        └── main.js      # JavaScript для інтерактивності
```

## 🚀 Як запустити проєкт локально

### Вимоги
- Python 3.7 або вище
- pip (менеджер пакетів Python)

### Крок 1: Встановлення залежностей

Відкрийте термінал у папці проєкту та виконайте:

```bash
pip install -r requirements.txt
```

### Крок 2: Запуск додатку

```bash
python app.py
```

### Крок 3: Відкриття сайту

Після запуску відкрийте браузер та перейдіть за адресою:

```
http://localhost:5000
```

## 🎨 Технології

- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Tailwind CSS (через CDN)
- **Animations**: AOS (Animate On Scroll)
- **Confetti**: canvas-confetti
- **Fonts**: Google Fonts (Playfair Display, Inter)

## 🎯 Інтерактивні елементи

1. **Кнопка "Натисни для свята"** — запускає ефект конфеті
2. **Картки спогадів** — клікніть, щоб переглянути прихований текст
3. **Кнопка "Прочитати послання"** — показує тепле привітання з ефектом друку

## 🎨 Кастомізація

### Зміна тексту повідомлення

Відредагуйте файл `static/js/main.js`, знайдіть змінну `messageText`:

```javascript
const messageText = "Ваш текст тут...";
```

### Зміна кольорів

Кольори можна змінити в `static/css/style.css` або через Tailwind класи в HTML шаблонах.

### Додавання нових спогадів

Відредагуйте файл `templates/index.html` та додайте нові картки в секції галереї.

## 🌟 Розгортання на Render

### Найпростіший спосіб через GitHub

1. **Завантажте проєкт на GitHub:**
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/your-username/site-for-dad.git
git push -u origin main
```

2. **Створіть новий Web Service на Render:**
   - Перейдіть на [render.com](https://render.com)
   - Натисніть "New +" → "Web Service"
   - Підключіть ваш GitHub репозиторій
   - Render автоматично знайде `render.yaml` і налаштує все

3. **Налаштування (якщо потрібно):**
   - Name: `fathers-day-site` (або будь-яке ім'я)
   - Region: найближчий до вас
   - Plan: Free
   - Натисніть "Create Web Service"

### Альтернативний спосіб (без render.yaml)

Якщо ви не хочете використовувати `render.yaml`, налаштуйте вручну:

1. У налаштуваннях Render Web Service:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn app:app`
   - **Environment Variables**: не потрібно (використовується стандартний PORT)

### Перевірка деплою

Після деплою Render надасть URL вашого сайту (наприклад: `https://fathers-day-site.onrender.com`). Сайт буде доступний за цим посиланням.

### Логи та моніторинг

- Переглядайте логи в панелі Render
- Сайт автоматично перезапускається при помилках
- Free план має обмеження по часу бездіяльності (15 хвилин)
